# Learning Tools Plugin

Interactive teaching skills with chunked explanations, comprehension checks, and interrupt handling.

## Installation

```bash
/plugin install learning-tools@skilled-intelligence
```

## Skills

### explainer

Interactive explanation skill that breaks down complex topics into digestible chunks.

**Triggers:** "explain this", "help me understand", "break this down", "teach me about"

### threaded-explainer

Multi-threaded explanation skill that maintains context across explanation sessions with comprehension checks.

**Triggers:** "continue explaining", "let's dive deeper", "I have a follow-up question"

**Includes:** `references/output-examples.md` with example outputs

## Use Cases

- Learning new codebases
- Understanding complex algorithms
- Onboarding to unfamiliar domains
- Breaking down technical concepts
