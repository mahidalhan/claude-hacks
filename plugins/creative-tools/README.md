# Creative Tools Plugin

Creative output skills for diagrams, UI design, generative art, and visual explanations.

## Installation

```bash
/plugin install creative-tools@skilled-intelligence
```

## Skills

### mermaid-diagrams

Publication-quality Mermaid diagram generation for documentation and architecture visualization.

**Triggers:** "create a diagram", "visualize this flow", "diagram the architecture", "mermaid"

### frontend-design

Distinctive frontend design skill that produces unique, opinionated UI implementations.

**Triggers:** "design a frontend", "create a UI", "build a page", "design this interface"

### algorithmic-art

Generative art creation using algorithmic patterns and mathematical foundations.

**Triggers:** "create generative art", "algorithmic pattern", "procedural design"

**Includes:** `templates/` with starter patterns

### ascii-art-explainer

Visual explanations using ASCII art for terminal-friendly documentation.

**Triggers:** "ascii diagram", "text visualization", "terminal-friendly diagram"

## Use Cases

- Architecture documentation
- User flow visualization
- Unique UI prototypes
- Visual explanations in code comments
